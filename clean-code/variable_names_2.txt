6.1 Подходящий уровень абстракции

- В задаче "Сигналы НЛО" из курса 28 задач функция должна возвращать
массив десятичных чисел, поэтому результирующий массив логичнее
назвать decimal_nums вместо result

- Задача "Разблокировка мобильных телефонов".
Результат работы функции логичнее назвать не result_str, а unlock_sequence_keys,
поскольку функция возвращает последовательность кнопок для разблокировки.

- Задача "Восстановление таблицы зарплат".
Назовём список номеров сотрудников ids - employee_numbers, 
список зарплат сотрудников salary - employee_salaries.

- Количество комментариев группы VK
count -- count_comments_of_current_group.

- Длительность временной серии
current_sum -- duration_time_series.

6.2 Имена из пространства решения

binary_search // имя функции для своей реализации бинарного поиска
kernel_equation // ядро уравнения, которое будет решаться методами интегральных уравнений
slae_preconditioner // предобусловливатель для СЛАУ
authorization_code // код авторизации, например, при использовании протокола OAuth
child_node // дочерний узел в дереве

6.3 Контекст. Примеры из простой системы работы с документами, используется класс Document.

document_delete // функция удаления документа
document_create_date // поле создания документа
document_author // автор документа

6.4 Оптимальная длина имени.
