7.3
Вместо i j k более выразительные имена присваиваю в случае, когда
реализуется достаточно сложный алгоритм, к примеру, многоуровневый вариант алгоритма быстрых мультиполей,
в котором из ребер счетной сетки строится так называемое дерево уровней.

Каждый счетный шаг должен проходить по группам уровня, по дочерним группам каждой группы и 
по элементам этих групп.

Тогда вместо цикла по i j k, получается примерно следующее:

for group in range(level.groups_count):
    for child_group in range(level.groups[group].children_count):
        for elem in range(child_level.groups[child_group].count_elements)
            ...
